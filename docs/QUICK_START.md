---

## **🚀 Getting Started (3 Simple Steps)**

### **Step 1: Start the System**
```bash
node start.js
```

### **Step 2: Tell Us About Your Project**
The system will ask you a few simple questions:
- **Project name** (e.g., "my-awesome-app")
- **Description** (e.g., "A web app that helps users manage tasks")
- **Project type** (web app, mobile app, e-commerce, etc.)

### **Step 3: Let the AI Team Work**
Once you confirm your project details, the AI team will:
1. 📝 Analyze your requirements
2. 🏗️ Create a development plan
3. 👨‍💻 Start building your project
4. ✅ Ask for your approval at key milestones

## **🔧 Managing Your Project**

### **Check Project Status**
```bash
cd your-project-name
node ../setup.js
```

### **View Available Commands**
```javascript
await setup.help()
```

### **Check Project Progress**
```javascript
await setup.status()
```

### **View Pending Approvals**
```javascript
await setup.pending()
```

### **Approve Work**
```javascript
await setup.approve('task-id')
```

## **🚨 Troubleshooting**

### **"Module not found" errors**
Make sure you're running from the correct directory and all files are present.

### **"Permission denied" errors**
Check that you have write permissions in the current directory.

### **Project won't start**
Try running `node start.js` again, or check the logs in `.teamleader/` folder.
